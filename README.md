# AI Resume Generator / AI 求職投遞助手

AI-powered job application assistant built with Streamlit.

使用 AI 協助生成求職信、優化履歷與面試準備建議的小工具。

## Features / 功能

- Generate job application letters / 生成求職信
- Resume improvement suggestions / 履歷優化建議
- Interview preparation tips / 面試準備建議
- Upload resume and paste job description / 上傳履歷並貼上職缺描述
